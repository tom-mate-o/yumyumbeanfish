This site is statically built from Markdown with Python
([Jinja2](https://jinja.palletsprojects.com/) + [Python-Markdown](https://python-markdown.github.io/))
and hosted on [GitHub Pages](https://pages.github.com/). So there is no CMS.

Photos are pulled automatically from my [Glass](https://glass.photo/) RSS feed,
records from my [Discogs](https://www.discogs.com/) collection — both refreshed
every two days via GitHub Actions.

Typeface: [Space Mono](https://fonts.google.com/specimen/Space+Mono) by Colophon Foundry, via Google Fonts.

Contactform:  [Web3Forms](web3forms.com)
